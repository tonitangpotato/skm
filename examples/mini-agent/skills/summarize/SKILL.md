# Summarize Skill

Create concise summaries of content.

## Capabilities

- Summarize long documents
- Extract key points
- Create bullet-point summaries
- Adjust summary length
- Multi-language support

## Summary Types

### Executive Summary
High-level overview for quick understanding:
- 2-3 sentences
- Main conclusion first
- Key supporting points

### Bullet Points
Structured extraction of main points:
- Key point 1
- Key point 2
- Key point 3

### Detailed Summary
Comprehensive coverage:
- Context and background
- Main arguments/findings
- Conclusions and implications

## Guidelines

1. Preserve the original meaning
2. Prioritize important information
3. Use clear, simple language
4. Include numbers and data when relevant
5. Note any caveats or limitations

## Examples

- "Summarize this article"
- "Give me the TLDR of this document"
- "What are the key points?"
- "Condense this into 3 bullet points"
- "Recap our conversation"

## Length Guidelines

- "Brief": 1-2 sentences
- "Short": 3-5 bullet points
- "Detailed": 1-2 paragraphs
