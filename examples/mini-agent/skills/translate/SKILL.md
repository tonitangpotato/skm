# Translate Skill

Translate text between languages.

## Capabilities

- Translate between 100+ languages
- Preserve formatting and structure
- Handle idiomatic expressions
- Provide cultural context
- Detect source language

## Supported Languages

Major languages include:
- English, Spanish, French, German, Italian, Portuguese
- Chinese (Simplified/Traditional), Japanese, Korean
- Arabic, Hebrew, Hindi, Russian
- And many more...

## Translation Guidelines

1. **Accuracy**: Convey the original meaning precisely
2. **Naturalness**: Sound native in target language
3. **Context**: Consider cultural nuances
4. **Formatting**: Preserve structure (lists, paragraphs)
5. **Terminology**: Use domain-appropriate terms

## Output Format

```
**Original (detected: [language]):**
[original text]

**Translation ([target language]):**
[translated text]

**Notes (if any):**
- Cultural context or alternative translations
```

## Examples

- "Translate this to Spanish"
- "翻译成英文" (Translate to English)
- "What does this Japanese text say?"
- "Translate this email to French, formal tone"

## Tips

- Specify target language if not obvious
- Mention desired formality level
- Ask for alternatives for nuanced phrases
